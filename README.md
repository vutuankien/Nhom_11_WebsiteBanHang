# Website bán đồ ăn Lotteria
- Chức năng:
    Client:
        + Đặt hàng
        + Tìm kiếm
        + Đăng nhập , Đăng ký
        + Mua hàng
        + Chat với quán (Dự tính)
    Admin:
        + Thêm Hàng
        + Xóa Hàng
        + Tìm Kiếm hàng
        + Cập nhật
        + Theo dõi đơn hàng
        + Chat với Khách hàng ( Dự tính )

Công nghệ : 
    - ReactJs, Bootstrap5, Json-server
